# Handoff document template

Use this structure for the `/tmp/handoff-<YYYY-MM-DD-HHMMSS>.md` file.

```markdown
# Handoff — <date>

## Task
[What was being done and why — 2–3 sentences]

## Status
- Done: [...]
- In progress: [...]
- Blocked: [...] (if any)

## Key decisions
[Non-obvious choices made with rationale — most valuable part, never truncate]

## Files changed
- `path/to/file` — [one-line purpose]

## Next steps (ordered, max 7)
1. [Most urgent first]

## Suggested skills
- `/skill-name` — [why]
```
